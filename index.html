Que es JavaScript

JavaScript (\(JS\)) es un lenguaje de programación que se usa para hacer páginas web interactivas y dinámicas, trabajando junto con HTML y CSS para crear la estructura, el estilo y la funcionalidad de un sitio. Permite desarrollar desde animaciones y carruseles de imágenes hasta aplicaciones web completas, juegos, validación de formularios y la actualización de contenido sin recargar la página.

¿Cómo funciona?
  • Se ejecuta directamente en el navegador del usuario, lo que se conoce como "lenguaje del lado del cliente".
  • Cuando un usuario visita una página web, su navegador descarga y ejecuta el código JavaScript, que puede estar incrustado en la página o en un archivo separado.
  • Este proceso crea una experiencia de usuario más dinámica, ya que el navegador puede procesar el código y cambiar el contenido en tiempo real, sin necesidad de volver a contactar al servidor.
¿Para qué se utiliza?
  • Interactividad: Añade elementos dinámicos como menús desplegables y la respuesta a clics de botones.
  • Animaciones y efectos: Permite crear animaciones, mover elementos en la página, o cambiar colores.
  • Actualización de contenido: Muestra nuevo contenido sin tener que recargar toda la página, como en las redes sociales.
  • Validación de formularios: Verifica que la información ingresada por el usuario en un formulario sea correcta antes de enviarla al servidor.
  • Aplicaciones web: Se utiliza para construir desde aplicaciones web hasta aplicaciones móviles.
Diferencias con Java
  • Es importante no confundir JavaScript con Java, ya que son dos lenguajes distintos.
  • Java es un lenguaje compilado, de tipado estático y más estructurado.
  • JavaScript es un lenguaje interpretado, de tipado dinámico y con una sintaxis menos estricta, lo que lo hace más flexible.

Comandos JavaScript

Los "comandos" de JavaScript no existen como tales, sino que se refiere a instrucciones, sentencias, operadores y funciones. Algunas de las instrucciones más comunes son la asignación de valores a variables (=), la declaración de funciones (function) y el uso de operadores lógicos (===, !==, &&, ||, !). También existen comandos específicos, como los de edición de texto en HTML5 (createLink, cut, delete) y los que se usan en la línea de comandos o en la consola del navegador.

Tipos de comandos en JavaScript

  • Operadores: Se utilizan para realizar operaciones.
    • Asignación: =.
    • Aritméticos: +, -, *, /.
    • Comparación: === (igual estricto), !== (no igual estricto), < (menor que),> (mayor que).
  • Sentencias de control: Controlan el flujo de ejecución.
    • Bucles: for, while, do...while.
    • Control de bucles: break, continue.
  • Funciones: Bloques de código reutilizables.
    • Se pueden definir de varias maneras, siendo la más común la declaración function nombre(parámetros) { }.
  • Comandos de edición de texto (HTML5): Se usan con el método document.execCommand() para manipular contenido rico.
    • createLink: Crea un hiperenlace sobre el texto seleccionado.
    • cut: Corta el texto seleccionado y lo copia al portapapeles.
    • delete: Borra el texto seleccionado.
    • fontName: Cambia la fuente del texto.
  • Comandos de consola: Se utilizan en la consola del navegador para interactuar directamente con JavaScript.
    • Se accede a ella con Ctrl+Shift+J (Windows, Linux) o Cmd+Option+J (macOS).
    • Permite ejecutar sentencias, como console.log("hola"), y ver los resultados.

Estructura básica

  • Punto y coma (;): Cada instrucción debe terminar con un punto y coma.
  • Comentarios: Se usan para explicar el código.
    • // para comentarios de una sola línea.
    • /* ... */ para comentarios de varias líneas.
  • Etiquetas script: Para incluir código JavaScript dentro de un archivo HTML, se utiliza la etiqueta script.

Historia de JavaScript

JavaScript se creó en 1995 por Brendan Eich en Netscape y originalmente se llamó "Mocha" y luego "LiveScript". Fue rebautizado como "JavaScript" como estrategia de marketing para capitalizar la popularidad de Java, a pesar de las diferencias significativas entre los lenguajes. El objetivo inicial era añadir interactividad dinámica a las páginas web en el navegador Netscape Navigator. La estandarización llegó en 1997 con la versión ECMAScript 1.
